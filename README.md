# trading_be
Trading API

## How To Run?
````
 go run ./bin/app/main.go ./bin/app/server.go
````

### Keep Run While Developing?

#### Install Nodemon
````
 npm i -g nodemon
````
#### Run
````
 nodemon --exec go run ./bin/app/main.go ./bin/app/server.go
````


## Some Module Error?
````
  go mod tidy
````

## Want To Tidy Up Your Code?
````
  gofmt -s -w .
````
